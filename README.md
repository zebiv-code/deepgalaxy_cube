# Deep Galaxy Cube

A rotating cube with a deep galaxy shader on each face, floating in a procedural starfield with nebulae. WebGL2 with custom GLSL shaders.

**[Live Demo](https://zebiv.com/deepgalaxy-cube/)**

## Tech

- WebGL2 (no libraries)
- Custom GLSL fragment shaders for cube faces and galaxy background
- Cube face shader based on work by [@YoheiNishitsuji](https://x.com/YoheiNishitsuji/status/1930972952318685444)
- Single HTML file, no build step
